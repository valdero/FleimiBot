# fleimi-bot

Telegram bot to release a plethera of hot fire towards an unsuspecting target.

## Current features
* /bomb - releases a single bomb
* /serve - bombs and 'dank' memes
* /rekt - gets a random drop of rekt
* /mindblown - used when ones mind is blown
